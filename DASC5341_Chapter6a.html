6.1 Simple Linear regression¶

Simple linear regression is a statistical method used to predict a continuous response variable from a single explanatory variable. For example, consider the cars data in R, where speed represents a car’s speed and dist represents its stopping distance.

If a car were selected at random, one simple guess for its stopping distance would be the overall mean stopping distance for all cars in the dataset. This might be a reasonable starting point if no other information were available. However, such a guess would often be inaccurate because cars traveling at different speeds are expected to have different stopping distances.

Since stopping distance is related to speed, we can improve prediction by using speed as an explanatory variable. A statistical method that takes advantage of this relationship is simple linear regression.

The term simple linear regression means that the model includes only one explanatory variable. In this example, speed is the explanatory variable, and dist is the response variable. The goal is to describe how stopping distance changes with speed and to use that relationship to predict stopping distance for a given speed.

The statistical model for simple linear regression is

$$ Y_i = \beta_0 + \beta_1 X_i + \varepsilon_i, \qquad i = 1,2,\dots,n, $$

where:

  • $Y_i$ is the response variable for the $i$th observation,
  • $X_i$ is the explanatory variable for the $i$th observation,
  • $\beta_0$ is the intercept,
  • $\beta_1$ is the slope,
  • $\varepsilon_i$ is the random error term.

Equivalently, the model can be written in terms of the conditional mean:

$$ E(Y_i \mid X_i) = \beta_0 + \beta_1 X_i. $$

This equation means that the average value of the response variable is assumed to change linearly with the explanatory variable.

Interpretation of the parameters¶

The intercept, $\beta_0$, is the mean value of the response variable when $X = 0$. In other words, it is the point where the regression line crosses the vertical axis. In some applications, this value has a practical interpretation; in others, it mainly serves to define the regression line.

The slope, $\beta_1$, represents the change in the mean response associated with a one-unit increase in the explanatory variable. If $\beta_1 > 0$, the mean response increases as $X$ increases. If $\beta_1 < 0$, the mean response decreases as $X$ increases. If $\beta_1 = 0$, then the explanatory variable does not help explain the mean response.

The error term, $\varepsilon_i$, represents the difference between the observed response and the mean response predicted by the regression line. It captures the variation in the response that is not explained by the explanatory variable alone.

Assumptions of simple linear regression¶

Simple linear regression is based on the following assumptions:

  1. Linearity
    The mean of the response variable is a linear function of the explanatory variable: $$ E(Y_i \mid X_i) = \beta_0 + \beta_1 X_i. $$

  2. Independence (how the data were collected)

    The observations are independent of one another. Equivalently, the error terms are assumed to be independent.

  3. Constant variance
    The error terms have the same variance for all values of the explanatory variable: $$ \operatorname{Var}(\varepsilon_i) = \sigma^2. $$ This assumption is also called homoscedasticity.

  4. Normality of errors
    The error terms are normally distributed: $$ \varepsilon_i \sim N(0,\sigma^2). $$ This assumption is mainly needed for statistical inference, such as confidence intervals and hypothesis tests.

For example, consider the cars data in R, where speed represents a car’s speed and dist represents its stopping distance. The regression model is $$ dist_i = \beta_0 + \beta_1 speed + e_i. $$

Estimation of parameters and prediction¶

In a simple linear regression model, the unknown parameters $\beta_0$ and $\beta_1$ are typically estimated by the least-squares method. The resulting estimates are denoted by $\hat{\beta}_0$ and $\hat{\beta}_1$, and they are called the least-squares estimators. The basic idea is intuitive: among all possible straight lines that could be drawn through the data, we choose the one that fits the observed points best.

To explain what “best” means, consider the difference between an observed response value and the value predicted by a fitted line. This difference is called a residual. Some residuals are positive and some are negative, so simply adding them would allow positive and negative values to cancel out. To avoid this, the least-squares method squares the residuals and then adds them together. The line chosen is the one that makes this total squared error as small as possible. In this sense, the least-squares estimators produce the fitted line that stays as close as possible to the observed data overall.

The least-squares estimators are

$$ \hat{\beta}_1 = \frac{\sum_{i=1}^n (X_i-\bar{X})(Y_i-\bar{Y})} {\sum_{i=1}^n (X_i-\bar{X})^2}, \qquad \hat{\beta}_0 = \bar{Y} - \hat{\beta}_1 \bar{X}. $$

(1)

Using these estimates, the fitted regression line is

$$ \hat{Y}_i = \hat{\beta}_0 + \hat{\beta}_1 X_i. $$

The least-squares estimators are chosen to minimize the sum of squared residuals,

$$ SSE = \sum_{i=1}^n (Y_i-\hat{Y}_i)^2. $$

Once the regression parameters have been estimated, the fitted line can be used for prediction. If the explanatory variable takes the value $x$, then the predicted mean response is

$$ \hat{Y}(x) = \hat{\beta}_0 + \hat{\beta}_1 x. $$

(2)

This is the estimated value of $E(Y \mid X=x)$, so $\hat{Y}(x)$ represents the estimated average response for all observations having explanatory variable value $x$.

Residual variance¶

The residual for the $i$th observation is

$$ e_i = Y_i - \hat{Y}_i. $$

The residual sum of squares is

$$ SSE = \sum_{i=1}^n e_i^2. $$

An unbiased estimator of the error variance $\sigma^2$ is

$$ s^2 = \frac{SSE}{n-2}, $$

and the corresponding residual standard error is

$$ s = \sqrt{\frac{SSE}{n-2}}. $$

This quantity measures the typical size of the residuals and describes how far the observed responses tend to fall from the fitted regression line.

Confidence interval for the mean response at $x$¶

A confidence interval at $x$ is used when the goal is to estimate the mean response $E(Y \mid X=x)$. In other words, it gives a plausible range for the average value of the response variable when the explanatory variable equals $x$.

The standard error of $\hat{Y}(x)$ is

$$ SE\big(\hat{Y}(x)\big) = s \sqrt{ \frac{1}{n} + \frac{(x-\bar{X})^2}{\sum_{i=1}^n (X_i-\bar{X})^2} }. $$

Therefore, a $100(1-\alpha)\%$ confidence interval for the mean response at $x$ is

$$ \hat{Y}(x) \pm t_{\alpha/2,\;n-2} \, s \sqrt{ \frac{1}{n} + \frac{(x-\bar{X})^2}{\sum_{i=1}^n (X_i-\bar{X})^2} }. $$

(3)

This interval reflects the uncertainty in estimating the regression line and, therefore, the uncertainty in estimating the mean response at the given value $x$.

Prediction interval for a new response at $x$¶

A prediction interval at $x$ is used when the goal is to predict a single future observation rather than the mean response. Since an individual observation can vary around the regression line, a prediction interval must account for both the uncertainty in estimating the mean response and the additional random variation of individual outcomes.

The standard error for prediction is

$$ SE_{\text{pred}} = s \sqrt{ 1 + \frac{1}{n} + \frac{(x-\bar{X})^2}{\sum_{i=1}^n (X_i-\bar{X})^2} }. $$

Thus, a $100(1-\alpha)\%$ prediction interval for a new response at $x$ is

$$ \hat{Y}(x) \pm t_{\alpha/2,\;n-2} \, s \sqrt{ 1 + \frac{1}{n} + \frac{(x-\bar{X})^2}{\sum_{i=1}^n (X_i-\bar{X})^2} }. $$

(4)

Because it includes the extra variability of an individual future response, a prediction interval is always wider than the corresponding confidence interval at the same value of $x$.

Meaning of the intervals¶

A confidence interval at $x$ describes a plausible range for the unknown mean response $E(Y \mid X=x)$. A prediction interval at $x$ describes a plausible range for a single future response value when $X=x$.

Thus, Equation (2) is the point estimate of the mean response, Equation (3) gives the confidence interval for the mean response, and Equation (4) gives the prediction interval for a new observation.

Correlation and coefficient of determination¶

Two important measures in simple linear regression are the correlation coefficient and the coefficient of determination. Both describe the relationship between the explanatory variable $X$ and the response variable $Y$, but they focus on different aspects of that relationship.

The sample correlation coefficient, denoted by $r$, measures the strength and direction of the linear relationship between two quantitative variables. Its value always lies between $-1$ and $1$. If $r>0$, then larger values of $X$ tend to be associated with larger values of $Y$. If $r<0$, then larger values of $X$ tend to be associated with smaller values of $Y$. If $r$ is close to $0$, then there is little or no linear relationship between the two variables. In general, values of $r$ close to $1$ or $-1$ indicate a strong linear relationship, while values close to $0$ indicate a weak linear relationship.

In simple linear regression, the sign of the slope estimator $\hat{\beta}_1$ is the same as the sign of the correlation coefficient $r$. Thus, if the slope is positive, the correlation is positive; if the slope is negative, the correlation is negative. However, correlation measures only linear association and does not imply causation.

The coefficient of determination, denoted by $R^2$, measures how well the regression model explains the variability in the response variable. It is defined by

$$ R^2=\frac{SSR}{SST}=1-\frac{SSE}{SST}, $$

where $SST$ is the total sum of squares, $SSR$ is the regression sum of squares, and $SSE$ is the error sum of squares. The value of $R^2$ always lies between $0$ and $1$. A value of $R^2$ close to $1$ means that the regression model explains a large proportion of the variation in the response variable, while a value close to $0$ means that the model explains little of that variation.

For example, if $R^2=0.80$, then about $80\%$ of the variability in $Y$ is explained by its linear relationship with $X$, and the remaining $20\%$ is due to other factors and random variation. Thus, $R^2$ tells us how useful the fitted regression line is for explaining the observed variation in the response.

In simple linear regression with one explanatory variable, the coefficient of determination is the square of the sample correlation coefficient:

$$ R^2=r^2. $$

This relationship shows that the two measures are closely connected. The correlation coefficient $r$ describes both the strength and the direction of the linear relationship, whereas $R^2$ describes the proportion of variation in the response explained by the model. Because $R^2$ is a squared quantity, it does not indicate direction. For instance, both $r=0.9$ and $r=-0.9$ give $R^2=0.81$.

Hypothesis testing for the slope and intercept¶

In simple linear regression, we often want to test whether the slope $\beta_1$ or the intercept $\beta_0$ differs from a specified value. More generally, we may test whether a parameter is equal to, greater than, or less than some constant.

For the slope, suppose we want to test

$$ H_0:\beta_1=\beta_{1,0} \qquad \text{versus} \qquad H_a:\beta_1\ne \beta_{1,0}, $$

or one of the one-sided alternatives

$$ H_a:\beta_1>\beta_{1,0} \qquad \text{or} \qquad H_a:\beta_1<\beta_{1,0}. $$

The test statistic is

$$ t=\frac{\hat{\beta}_1-\beta_{1,0}}{SE(\hat{\beta}_1)}, $$

where

$$ SE(\hat{\beta}_1)= \frac{s}{\sqrt{\sum_{i=1}^n (X_i-\bar{X})^2}}, $$

and $s^2=SSE/(n-2)$. Under the null hypothesis, this test statistic follows a $t$ distribution with $n-2$ degrees of freedom.

The same idea applies to the intercept. To test

$$ H_0:\beta_0=\beta_{0,0} \qquad \text{versus} \qquad H_a:\beta_0\ne \beta_{0,0}, $$

or one-sided alternatives, we use the test statistic

$$ t=\frac{\hat{\beta}_0-\beta_{0,0}}{SE(\hat{\beta}_0)}, $$

where

$$ SE(\hat{\beta}_0)= s\sqrt{ \frac{1}{n} + \frac{\bar{X}^2}{\sum_{i=1}^n (X_i-\bar{X})^2} }. $$

Again, under the null hypothesis, this statistic has a $t$ distribution with $n-2$ degrees of freedom.

A common special case is testing whether the slope is zero:

$$ H_0:\beta_1=0 \qquad \text{versus} \qquad H_a:\beta_1\ne 0. $$

This test examines whether the explanatory variable contributes to explaining the response. If $H_0$ is rejected, then there is evidence that the response and explanatory variable have a linear relationship. The interpretation depends on whether $X=0$ is meaningful in the problem. In many applications, the slope is of greater practical interest than the intercept.

Model checking¶

After fitting a simple linear regression model, it is important to check whether the model assumptions are reasonable. Model checking helps us determine whether the fitted regression line provides an appropriate description of the relationship between the explanatory variable and the response variable. If the assumptions are not satisfied, then the estimates, tests, confidence intervals, and predictions from the model may not be reliable.

A useful first step is to examine a scatterplot of the response variable versus the explanatory variable before fitting the model. The scatterplot helps us assess whether the relationship appears approximately linear. If the points follow a roughly straight-line pattern, then a simple linear regression model may be appropriate. On the other hand, if the pattern is curved or highly irregular, then a linear model may not fit the data well. After fitting the model, it is also helpful to add the fitted regression line to the scatterplot to see whether the line captures the overall trend in the data.

Another important step is to examine the residuals, which are the differences between the observed values and the fitted values. A plot of residuals versus the fitted values, or residuals versus the explanatory variable, can be used to check whether the variance appears roughly constant and whether there are signs of nonlinearity. Ideally, the residuals should be randomly scattered around zero with no clear pattern. If the spread of the residuals increases or decreases as the fitted values increase, then the constant variance assumption may be violated. If a curved pattern appears, then the linearity assumption may not hold.

The normality assumption is commonly checked by using a histogram of residuals, a normal Q-Q plot, or a formal normality test such as the Shapiro-Wilk test. If the residuals are approximately normal, then the points in the Q-Q plot should lie close to a straight line. The Shapiro-Wilk test can also be used to test the null hypothesis that the residuals come from a normal distribution. A large p-value suggests that the normality assumption is reasonable, while a very small p-value suggests departure from normality. However, the normality test should be interpreted together with the plots, since formal tests can be sensitive to sample size.

It is also important to check for unusual observations, such as outliers or influential points. An outlier is an observation with an unusually large residual, while an influential observation is one that has a strong effect on the fitted regression line. Such points can sometimes distort the estimated relationship and affect the regression results. Diagnostic plots and influence measures can help identify these observations for further investigation.

In addition, if the data are collected over time or in sequence, the independence of the errors should be considered. For time-ordered data, residuals may be correlated with one another, which violates a basic regression assumption. In such cases, plotting the residuals in their data collection order may help reveal patterns or dependence. If strong dependence is present, a different modeling approach may be needed.

Overall, model checking is an essential part of regression analysis. A fitted model should not be accepted only because the regression equation has been obtained. Instead, graphical and numerical diagnostics should be used to verify that the assumptions of linearity, constant variance, normality, and independence are reasonably satisfied.

Multiple Regression¶

The multiple regression model relates one response variable to several predictors. It is commonly written as

$$ Y_i=\beta_0+\beta_1 x_{i1}+\beta_2 x_{i2}+\cdots+\beta_p x_{ip}+\varepsilon_i, \qquad i=1,2,\dots,n. $$

Here:

  • $Y_i$ is the response for observation $i$,
  • $x_{i1},x_{i2},\dots,x_{ip}$ are the predictor values for observation $i$,
  • $\beta_0$ is the intercept,
  • $\beta_1,\beta_2,\dots,\beta_p$ are regression coefficients,
  • $\varepsilon_i$ is the random error term.

The model says that the mean of $Y_i$, given the predictors, is

$$ E(Y_i \mid x_{i1},x_{i2},\dots,x_{ip}) =\beta_0+\beta_1 x_{i1}+\beta_2 x_{i2}+\cdots+\beta_p x_{ip}. $$

So the fitted regression equation is

$$ \hat{Y}_i=\hat{\beta}_0+\hat{\beta}_1 x_{i1}+\hat{\beta}_2 x_{i2}+\cdots+\hat{\beta}_p x_{ip}, $$

where $\hat{\beta}_0,\hat{\beta}_1,\dots,\hat{\beta}_p$ are estimates from the data.

Under the usual assumptions,

$$ \varepsilon_i \sim \text{independent } N(0,\sigma^2), $$

or more generally,

$$ E(\varepsilon_i)=0, \qquad \operatorname{Var}(\varepsilon_i)=\sigma^2. $$

Similiar assumptions for a simple linear regression are also needed for multiple regression.

anova¶

Question: can we remove biceps, forearm, and ankle from the model at the same time? Is the model with all variables significant better than the model without these three variables? We need to tests whether biceps, forearm, and ankle can be removed from the full model simultaneously.

$H_0: \beta_{bicep} = \beta_{forearm} = \beta_{ankle} =... = 0$ vs $H1: $ at least one is not zero.

Reduced model is the model under H0 with less variables.

Common diagnostic measures in multiple regression¶

1. Leverage¶

Leverage measures how unusual the independent values are for observation $i$. Large leverage means the observation is far from the center of the predictor space and may strongly affect the fitted model.

A common rule of thumb is to examine observations with $h_{ii} > \frac{2p}{n}$

where $p$ is the number of parameters in the model, including the intercept, and $n$ is the sample size.

2. Studentized residuals¶

Studentized residuals are used to detect outliers in the response variable. Large absolute values suggest that the observed response is unusual relative to the fitted model.

A common guideline is to check observations with

$$ |t_i| > 3. $$

3. Cook’s distance¶

Cook’s distance measures the overall influence of an observation on the fitted regression model. It combines information from both the residual size and the leverage.

Large values of $D_i$ suggest that removing observation $i$ may noticeably change the fitted model.

Common rough guidelines are to examine observations with $D_i > \frac{4}{n}$.

4. Variance inflation factor (VIF)¶

VIF measures how strongly predictor $X_j$ is linearly related to the other predictors. Large VIF values indicate multicollinearity, which inflates the variance of the estimated coefficient $\hat{\beta}_j$.

A common interpretation is:

  • $\mathrm{VIF} > 5$: potentially problematic
  • $\mathrm{VIF} > 10$: serious multicollinearity
